1. 인간다운 언어다.

파이썬 기초문법
1. 사칙연산
덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/)

2. 변수에 대입하고 출력하기
a="python" : 변수에 대입
pritn(a) :변수 a출력
결과:python

3. 조건문 if
a=3
if a>1: #if (조건식):
    print("a is greater than 1!") #들여쓰기후 수행할 문장
결과: a is greater than 1!
 만약 a가 1보다 크다면 "a is greater than 1!"이란 문장 출력해라.
이때, if 다음 문장은 tab or 띄어쓰기 4번을 이용해 반드시 들여쓰기한다.

4. 반복문 while
i=0
while i < 3: #while (조건식):
    i=i+1
    print(i)
결과:
1
2
3

while이란 단어는 "~동안" 즉, i가 3미만인 동안 i에1씩 더하고 출력해라.
를 반복하다 값이 3보다 커지면 while문을 빠져나간다.

5. 반복문 for
for a in [1,2,3]:
    print(a)
결과: 
1
2
3
 
[1,2,3]리스트의 요소들을 순차적으로 꺼내서 a에 대입 후 출력해라를 "반복!"

6. 함수
def add(a,b): #define add란 함수를 (a,b)에 관해 정의한다.
    return a+b #출력 a+b를 출력
a= add (3,4)
print(a)
결과: 7

파이썬에서 def는 함수를 만들때 사용하는 예약어로
add함수를 만들고 그함수를 어떻게 사용하는지 보여준다. add(a,b)는 입력값이고, a+b는 결괏값이다.
즉 3,4 입력이 들어오면 3+4를 수행하고 그 결괏값인 7을 돌려준다.(return)
