# 1장 연습문제
## Q12
파이썬은 다음처럼 동일한 값에 여러 개의 변수를 선언할 수 있다. 다음과 같이 a,b 변수를 선언한 후 a의 두번째 요솟값을 변경하면 b 값은 어떻게 될까? 그리고 이런 결과가 오는 이유에 대해 설명해 보자.

---
```python
>>> a=b=[1,2,3]
>>> a[1]=4
>>> print(b)
```

```python
a=b=[1,2,3]
a[1]=4
print(b)
print(id(a))
print(id(b))
1930409194176
1930409194176
```
a와b 는 같은 리스트 주소값을 바라보고있다. 따라서 a를 수정해도 b값도 같이 수정된 결과가 나오는 것이다.
